# FEATURES.md

## 目标功能清单

本项目长期目标是形成一个可工程化运行的 **OpenAI 兼容 API / 账号池管理 / 自动化运维系统**。

---

## 核心目标能力

1. **OpenAI 兼容 API 网关**
   - 暴露统一请求入口
   - 支持模型列表与调用转发

2. **账号池管理**
   - 账号录入
   - OAuth 接入
   - 账号状态管理
   - 账号健康检查

3. **调度与路由**
   - 请求路由
   - 限流/配额管理
   - 基础负载均衡
   - 可用账号优先选择

4. **自动化控制**
   - 定时巡检
   - 补号触发
   - 告警通知
   - 审计日志

5. **部署与运维**
   - Ubuntu + Docker 部署
   - 反向代理与 HTTPS
   - 配置管理
   - 运行状态查看

---

## 近期不应误判为已完成的能力

以下内容目前仍主要停留在方案层，不应视为已实现：

- 批量账号自动获取
- 完整 OAuth 自动化脚本
- 真正可运行的 API 网关代码
- 自动补号闭环
- 真实可用的监控告警系统
